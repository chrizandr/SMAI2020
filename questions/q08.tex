  \[ \min_{\bf w} (3w_1 + 4 w_2 - 12)^2 + \lambda g({\bf w})\]

  If \underline{$g({\bf w})$ is L0 norm of ${\bf w}$, and $\lambda =1$}, what is the optimal value of ${\bf w}$
     \begin{enumerate}[label=(\Alph*)]

      \item $[1,1]^T$
      \item $[3,4]^T$
      \item None of the above   % None
     \end{enumerate}
